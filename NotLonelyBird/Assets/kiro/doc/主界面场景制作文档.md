# 主界面场景制作文档

## 场景概述

主界面是应用的功能选择页面，展示 6 个功能入口按钮（记忆力、注意力、执行功能、情感识别、退出应用、虚实绘景），每个按钮以图标卡片形式排列，底部带文字标签。

场景路径：`Assets/kiro/主界面.unity`

---

## 场景层级结构

```
主界面.unity
├── Directional Light              — 方向光
├── RKCameraRig (Prefab)           — XR 相机 + AllSettingCtr 单例
├── [RKInput] (Prefab)             — XR 输入系统
└── PointableUI (Prefab)           — World Space Canvas
    ├── 记忆力 (Button + Image)
    │   └── Text (TMP) "记忆力"
    ├── 注意力 (Button + Image)
    │   └── Text (TMP) "注意力"
    ├── 执行功能 (Button + Image)
    │   └── Text (TMP) "执行功能"
    ├── 情感识别 (Button + Image)
    │   └── Text (TMP) "情感识别"
    ├── 虚实绘景 (Button + Image)
    │   └── Text (TMP) "虚实绘景"
    └── 退出应用 (Button + Image)
        └── Text (TMP) "退出应用"
```

## 业务逻辑层

### AllSettingCtr.cs（挂载在 RKCameraRig 上）

脚本路径：`Assets/kiro/Scripts/NBack/AllSettingCtr.cs`

全局单例，DontDestroyOnLoad，跨场景传递游戏设置参数。在主界面场景的 RKCameraRig 上首次创建。

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| nValue | int | 1 | N-Back 的 N 值 |
| stimulusDuration | float | 2.0 | 刺激显示时长（秒） |
| interStimulusInterval | float | 2.5 | 刺激间隔时长（秒） |
| totalTrials | int | 40 | 总试次数 |
| targetIndexN0 | int | 4 | N=0 模式目标位置（九宫格中心） |
| matchProbability | float | 0.33 | 匹配概率 |
| isOneBlockMode | bool | false | 单组模式开关 |

### InfiniteScrollPanel.cs

脚本路径：`Assets/kiro/Scripts/InfiniteScrollPanel.cs`

无限水平滚动面板（当前场景未使用 ScrollRect，但脚本已备用）。当内容滚动到边界时，将首/尾元素移到另一端实现循环。

### 场景切换逻辑

各功能按钮的 OnClick 事件目前为空（`m_Calls: []`），需要后续绑定 `ScenesChange.SceneChange()` 来跳转到对应子场景。

## UI 组件层

### PointableUI (World Space Canvas)

- Prefab GUID: `3c20833d81e354626b8365b459274912`
- Position: (0, 0, 0.5)
- 6 个功能按钮作为 AddedGameObjects 挂载到 Canvas 的 RectTransform 下

### 功能按钮通用结构

每个按钮由以下组件构成：

| 组件 | 说明 |
|------|------|
| RectTransform | 定位和尺寸 |
| CanvasRenderer | 渲染 |
| Image | 功能图标 Sprite |
| Button | 点击交互（Transition: ColorTint） |

按钮子物体：
- Text (TMP) — 功能名称标签

### 按钮布局参数

所有按钮共享：
- Scale: (0.7, 0.7, 0.7)
- SizeDelta: 198.1 × 300
- Anchor: 左上角 (0, 1)
- ConstrainProportionsScale: true

| 按钮名称 | AnchoredPosition | Sprite GUID |
|----------|-------------------|-------------|
| 记忆力 | (569, -531) | `20931516536db0f4a8baf4f5dafe7308` |
| 注意力 | (727.2, -531) | `5e6b42e6170c36547b22496511e5b13b` |
| 执行功能 | (885.8, -531) | `1464e248cc8a3fe41888601a3f33de10` |
| 情感识别 | (1044.1, -531) | `6d1b7576d948f6b44a8305bb913cfc2b` |
| 退出应用 | (1202.4, -531)* | 对应退出图标 |
| 虚实绘景 | (1360.7, -531)* | 对应虚实绘景图标 |

*退出应用和虚实绘景的精确位置按等间距排列推算
