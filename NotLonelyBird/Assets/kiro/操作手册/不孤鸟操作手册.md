# 不孤鸟 — 用户操作手册

## 1. 产品简介

"不孤鸟"是一款基于 AR 技术的儿童认知干预训练应用，运行于 Rokid XR 眼镜平台。系统基于剑桥 CANTAB 神经心理学范式，结合 Rokid 手势识别、千问 ASR 语音识别和 Tripo AI 3D生成技术，提供沉浸式的全感官统合训练体验。

### 1.1 核心功能模块

| 模块 | 场景名称 | 训练目标 |
|------|----------|----------|
| 记忆力训练 — 数字广度 | DigitSpan | 瞬时记忆、顺序/逆序复述 |
| 记忆力训练 — N-Back 回溯 | NBack | 空间工作记忆、信息更新 |
| 注意力训练 — 快速视觉处理 | RVP | 持续注意力、目标检测 |
| 执行力训练 — 方向判断 | Direction | 抑制控制、资源分配 |
| 情感认知训练 | Emotion | 情绪识别、社交认知 |
| 创意疗愈画板 | 3DGenerate | 创意表达、空间认知 |

### 1.2 运行环境

| 项目 | 要求 |
|------|------|
| 操作系统 | YodaOS-Master，基于 UXR 3.0 |
| 应用类型 | 6DoF AR 应用 |
| 处理器 | XR2 Gen1（最低） |
| RAM | 12GB（最低） |
| ROM | 128GB（最低） |

## 2. 启动与主界面

### 2.1 应用启动

佩戴 Rokid 眼镜后启动应用，首先进入品牌加载页面：

1. 屏幕显示"不孤鸟"应用图标，伴随短铃声音效
2. 图标渐显动画播放完毕后，自动跳转到主界面

### 2.2 主界面

主界面展示 6 个功能入口卡片，通过射线点击进入对应模块：

| 卡片 | 功能 | 跳转场景 |
|------|------|----------|
| 记忆力 | 数字广度 / N-Back 记忆训练 | 记忆力设置页 |
| 注意力 | 快速视觉信息处理 RVP | 注意力设置页 |
| 执行功能 | 方向判断多重任务测试 | 执行功能设置页 |
| 情感识别 | 3D卡通情感认知训练 | 情感识别场景 |
| 虚实绘景 | AI 创意疗愈画板 | 3DGenerate 场景 |
| 退出应用 | 关闭应用 | — |

操作方式：用射线对准卡片，捏合手势点击即可进入。

## 3. 记忆力训练

### 3.1 数字广度任务（DigitSpan）

训练瞬时记忆能力，系统播放一组随机数字音频，玩家需按要求复述。

#### 设置选项

进入前可在设置页调整：

| 设置项 | 选项 | 说明 |
|--------|------|------|
| 复述顺序 | 自动 / 顺序 / 逆序 | 自动模式下连续答对3次切换逆序 |
| 难度 | 随机 / 2位 / 3位 / 4位 / 5位 | 控制每轮数字位数 |

#### 操作流程

1. 点击"开始游戏"按钮
2. 系统播放模式提示音（"顺序"或"逆序"）
3. 系统依次播放数字音频（如"3-7-1"）
4. 屏幕显示"请回答"，此时可以作答

#### 两种答题方式

**方式一：按钮点击**
- 屏幕下方显示 0-9 数字按钮
- 按顺序（或逆序）点击听到的数字

**方式二：语音答题**
- 点击麦克风按钮开始录音
- 说出听到的数字
- 再次点击麦克风停止录音并提交

#### 反馈

- 回答正确：播放"正确"提示音
- 回答错误：播放"错误"提示音，显示正确答案
- 屏幕显示已完成题数

#### 特殊规则（自动模式）

- 连续答对 3 次 → 自动切换为逆序模式
- 答错 → 重置计数，回到顺序模式

---

### 3.2 N-Back 回溯

训练空间工作记忆，判断当前方块位置是否与 N 步之前相同。

#### 设置选项

| 设置项 | 范围 | 说明 |
|--------|------|------|
| N 值 | 0, 1, 2, 3... | N=0 判断是否在中心，N≥1 判断与N步前是否相同 |
| 刺激时长 | 秒 | 方块显示时间 |
| 刺激间隔 | 秒 | 两次方块之间的间隔 |
| 总试次 | 次 | 一轮游戏的总题数 |

#### 操作流程

1. 低头看向前方平面，可见 3×3 九宫格底板
2. 游戏开始后，方块依次出现在不同格子位置
3. 判断当前方块位置是否与 N 步之前相同
4. 如果匹配 → 点击"匹配时点击"按钮
5. 如果不匹配 → 不操作，等待下一个方块

#### 计分规则

| 情况 | 结果 |
|------|------|
| 匹配时点击了按钮 | 正确 +1 |
| 不匹配时未点击 | 正确 +1（正确拒绝） |
| 匹配时未点击 | 漏报 |
| 不匹配时点击了 | 虚报 |

游戏结束后显示正确率和得分。

## 4. 注意力训练 — 快速视觉处理（RVP）

高频呈现手势图片，精准捕捉特定目标序列。

#### 设置选项

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 每分钟手势数 | 40 | 刺激呈现速度 |
| 目标手势数量 | 3 | 需要记忆的目标手势个数 |

#### 操作流程

**阶段一：指令（5秒）**
1. 屏幕显示"以下图片出现时，模仿手势"
2. 展示本轮需要记忆的目标手势图片（随机选取）
3. 倒计时 5 秒

**阶段二：测试**
1. 手势图片快速轮播
2. 当看到目标手势时 → 做出对应手势（或按键盘模拟键）
3. 非目标手势 → 不操作
4. 屏幕显示 ✓ 或 ✗ 即时反馈

**阶段三：结果**
- 显示正确率、平均反应时间
- 显示命中/漏报/虚报/正确拒绝各项统计

#### 手势类型

| 手势 | 左手 | 右手 |
|------|------|------|
| 握拳 | ✓ | ✓ |
| 捏合 | ✓ | ✓ |
| 手掌朝前 | ✓ | ✓ |
| 手掌朝上 | ✓ | ✓ |

无 Rokid SDK 时可用键盘模拟：Q/W/E/R（左手）、U/I/O/P（右手）。

## 5. 执行力训练 — 方向判断（Direction）

左右视野分别呈现不同规则的箭头，训练大脑的抑制控制和资源分配。

#### 设置选项

| 设置项 | 说明 |
|--------|------|
| 每回合时长 | 回答时间限制 |
| 总回合数 | 一轮游戏的题数 |
| 随机模式 | 开启后箭头方向与出现位置不一致（更难） |

#### 操作流程

1. 点击"开始游戏"，保持双手握拳
2. 屏幕左侧或右侧出现一个箭头图标
3. 根据箭头指向的方向（不是箭头出现的位置）举起对应手掌
   - 箭头指左 → 举左手掌
   - 箭头指右 → 举右手掌
4. 倒计时内未作答视为错误

#### 难度说明

| 模式 | 说明 |
|------|------|
| 一致性模式 | 箭头方向 = 出现位置（较简单） |
| 随机模式 | 箭头方向与位置无关（需要抑制位置干扰） |

前 3 回合始终为一致性模式（热身），之后按设置切换。

无 Rokid SDK 时可用键盘：A = 左手，D = 右手。

## 6. 情感认知训练（Emotion）

利用 3D 卡通角色与动态微表情，在低焦虑环境中训练情感认知。

#### 设置选项

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 题目数量 | 10 | 一轮测试的总题数 |
| 表情显示时间 | 3 秒 | 每个表情的播放时长 |

#### 操作流程

1. 阅读介绍说明，点击"开始"
2. 3D 卡通角色（星星）出现在眼前
3. 角色播放一个情绪表情动画（持续数秒）
4. 表情播放结束后，角色回到待机状态
5. 屏幕出现 4 个情绪按钮：

| 按钮 | 对应情绪 |
|------|----------|
| 开心 | smiling |
| 伤心 | sad |
| 愤怒 | angry |
| 惊讶 | fear |

6. 选择你认为正确的情绪
7. 屏幕显示"正确"或"错误"
8. 自动进入下一题

#### 结算

游戏结束后显示：
- 本次正确率
- 历史最佳记录

## 7. 创意疗愈画板（3DGenerate / 虚实绘景）

利用 AI 将 2D 涂鸦实时转为 3D 实体，实现创意疗愈与空间认知训练。

#### 三种生成方式

**方式一：画板绘图 → 3D**
1. 在 AR 空间中的虚拟画板上绘画
2. 可通过工具栏修改画笔大小和颜色
3. 点击"生成"按钮，系统截取画板内容
4. Tripo AI 将画作转换为 3D 模型

**方式二：文字描述 → 3D**
1. 在文字输入框中输入描述（如"一只小猫"）
2. 点击"Generate"按钮
3. Tripo AI 根据文字描述生成 3D 模型

**方式三：语音输入 → 3D**
1. 点击麦克风按钮开始录音
2. 说出想要生成的物体描述
3. 再次点击停止录音
4. 系统通过 ASR 识别语音内容，自动转为文字并生成 3D 模型

#### 交互操作

| 操作 | 说明 |
|------|------|
| 捏一下"设置"按钮 | 弹出工具栏 |
| 捏住画笔 | 在画板上绘画 |
| 捏一下"生成"按钮 | 截取画板内容生成 3D 模型 |
| 抓取 3D 模型 | 生成完成后可用手势抓取、移动、旋转模型 |

#### 生成进度

生成过程中会显示进度条，完成后 3D 模型出现在场景中，可通过手势自由抓取和查看。

## 8. 通用操作说明

### 8.1 射线交互

所有 UI 界面均通过 Rokid 射线交互操作：
- 手指指向目标 → 射线瞄准
- 捏合手势 → 点击/确认
- 射线拖拽 → 移动可拖拽物体

### 8.2 返回导航

每个功能场景左上角均有"返回"按钮，点击可返回上一级页面或主界面。

### 8.3 设置页

记忆力、注意力、执行功能模块在进入训练前会经过设置页，可调整训练参数（难度、时长、模式等）。设置参数通过全局单例跨场景传递，无需重复配置。

---

## 9. 语音识别说明

系统集成千问 ASR 模型 API 实现语音识别，服务于数字广度任务和创意疗愈画板的语音输入功能。

| 项目 | 说明 |
|------|------|
| 采样率 | 16kHz |
| 最长录音 | 30 秒 |
| 服务器 | 隐私 |
| 格式 | WAV 文件上传，返回 JSON |

使用时确保网络连接正常，录音环境安静以提高识别准确率。

## 10. 场景导航总览

```
Load (启动页)
  └── 主界面
      ├── 记忆力 → MemoryReady (设置) → DigitSpan (数字广度)
      ├── 记忆力 → NBackReady (设置) → NBack (N-Back回溯)
      ├── 注意力 → AttentionReady (设置) → RVP (快速视觉处理)
      ├── 执行功能 → ExcutiveReady (设置) → Direction (方向判断)
      ├── 情感识别 → Emotion (情感认知)
      ├── 虚实绘景 → 3DGenerate (创意画板)
      └── 退出应用
```
